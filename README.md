# Tic Tac Toe

## Setup

### Requirements

- docker & docker-compose
- nodejs
- ember-cli

### Backend Setup (Rails)

- git clone git@github.com:shaliko/tic-tac-toe.git
- cd tic-tac-toe/game_api
- docker-compose up


### Frontend Setup (EmberJS)

- Project already cloned on previous steps
- cd tictactoe
- npm install
- ember s
